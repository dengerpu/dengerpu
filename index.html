• 查看版本 ---- git --version
  • 1.GIt使用前配置
    1. 配置提交人姓名--- git config --global user.name 提交人姓名
    2. 配置提交人邮箱--- git config --global user.email 提交人邮箱
    3. 查看配置信息--- git config --list
  • 2.提交配置
    1. 初始化仓库--- git init
    2. 查看文件状态--- git status
    3. 追踪文件--- git add 文件名称
    4. 向仓库提交代码--- git commit -m 提交信息
    5. 查看提交记录--- git log
  • 3撤销
    1. 用暂存区的文件覆盖工作目录的文件--- git checkout 文件名称
    2. 将文件从暂存区中删除--- git rm --cached 文件
    3. 将git仓库中指定的更新记录恢复出来，并且覆盖暂存工作目录--- git reset --hard id值
  • 4.分支命令（master主分支 develop开发分支）
    1. 查看分支--- git branch
    2. 创建分支--- git branch 分支名称
    3. 切换分支--- git checkout 分支名称
    4. 合并分支--- git merge 来源分支
    5. 删除分支--- git branch -d 分支名称 （分之合并后才允许删除）（-D强制删除）
