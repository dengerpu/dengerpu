• 查看版本 ---- git --version
  • 1.GIt使用前配置
    1. 配置提交人姓名--- git config --global user.name 提交人姓名
    2. 配置提交人邮箱--- git config --global user.email 提交人邮箱
    3. 查看配置信息--- git config --list
  • 2.提交配置
    1. 初始化仓库--- git init
    2. 查看文件状态--- git status
    3. 追踪文件--- git add 文件名称
    4. 向仓库提交代码--- git commit -m 提交信息
    5. 查看提交记录--- git log
